# Display Classes - Automatic.css

Display classes change the elementâ€™s [display property](https://developer.mozilla.org/en-US/docs/Web/CSS/display). They follow the syntax `.display--[value]`.

Options are:

*   inline
*   block
*   contents
*   inline-block
*   inline-flex
*   list-item
*   none

These classes are available at each breakpoint using the syntax, `.display--[breakpoint]-[value]`.
